JavaScript es responsable de la "capa interactiva" de una página web, que complementa la capa "estructural" que proporciona el lenguaje de marcado y la capa "presentacional" que proporciona CSS. JavaScript permite a los desarrolladores modificar la estructura y la presentación de una página agregando, quitando y alterando lenguaje de marcado y estilos en respuesta a cualquier combinación de interacción del usuario y lógica interna. JavaScript puede brindarte un gran nivel de control sobre el procesamiento y el comportamiento de una página web, y hasta te permite alterar algunos de los comportamientos integrados del navegador.
