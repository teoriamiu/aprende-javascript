Tipos y estructuras de datos Las primitivas son los tipos de datos más simples en JavaScript. Un literal primitivo es un valor, sin wrapper ni propiedades propias. Los literales primitivos son inmutables, lo que significa que no se pueden cambiar para representar otros valores de la misma manera que las estructuras de datos más complejas basadas en objetos de JavaScript. Por ejemplo, si bien al valor de una variable llamada theTruth se le puede reasignar un valor de false, el literal booleano true nunca puede representar ningún valor que no sea true, de la misma manera que el literal de número 5 nunca puede representar el valor de otro número. Existen siete tipos primitivos de datos: Números Cadenas Booleanos null undefined BigInt Símbolo Para obtener información sobre tipos de datos más complejos, consulta Variables.
